This is a Flysystem adapter for the aws-sdk-php v3.   This is a fork for use in a WordPress plugin.

# Installation

```bash
composer require ilab/ilab-flysystem-aws-s3-v3
```

# Bootstrap

``` php
<?php
use Aws\S3\S3Client;
use ILAB_League\Flysystem\AwsS3v3\AwsS3Adapter;
use ILAB_League\Flysystem\Filesystem;

include __DIR__ . '/vendor/autoload.php';

$client = new S3Client([
    'credentials' => [
        'key'    => 'your-key',
        'secret' => 'your-secret'
    ],
    'region' => 'your-region',
    'version' => 'latest|version',
]);

$adapter = new AwsS3Adapter($client, 'your-bucket-name');
$filesystem = new Filesystem($adapter);
```
